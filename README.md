# StreamVerse 🎬

**StreamVerse** is a modern universal web video player that allows users to stream videos from multiple sources and formats using a clean and responsive interface.

The player supports common video formats, adaptive streaming technologies, and cloud video sources — all within a single lightweight web application.

---

## ✨ Features

* 🎥 Universal video playback
* 🌐 Supports multiple video sources
* 📺 Adaptive streaming support
* ⚡ Lightweight and fast
* 🎨 Animated responsive UI
* 🧹 Clear input option
* 📱 Mobile friendly design

---

## 📦 Supported Formats

StreamVerse currently supports the following video formats and sources:

| Type                | Supported |
| ------------------- | --------- |
| MP4                 | ✅         |
| WebM                | ✅         |
| OGG                 | ✅         |
| HLS (.m3u8)         | ✅         |
| MPEG-DASH (.mpd)    | ✅         |
| Google Drive Videos | ✅         |

## ⚙️ Installation

Clone the repository:

```bash

```

Open the project folder:

```bash
cd streamverse
```

Then simply open **index.html** in your browser.
